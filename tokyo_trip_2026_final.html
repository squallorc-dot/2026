Day1｜2026/01/20（二） 抵達日

航班：TR874 桃園15:30 → 成田19:55 · 入住：Tobu Hotel Levant Tokyo（錦糸町）

🗺️ 查看飯店地圖
19:55
抵達成田 → 前往飯店

常見交通選擇：

  • Narita Express (N'EX)：直達東京、品川、上野等，行李多或想省換乘時方便（需預約/座位）。
  • Keisei Skyliner：最快到京成上野/日暮里，轉乘JR或地下鐵至錦糸町（較快但需轉車）。
  • Airport Limousine（利木津巴士）：直達大飯店與主要車站，省行李轉乘步驟（視時間點有班次）。

到東京後可搭 JR 或 Metro 轉乘至錦糸町，視航班時間與行李選擇最適合方案。

21:30
入住 & 晚餐

錦糸町周邊商店街、拉麵或居酒屋選擇多，可在飯店放行李後步行覓食。

飯店地址：1 Chome-2-2 Kinshi, Sumida City, Tokyo 130-0013

航班/價格參考（Skyscanner 等） 若要臨時更改航班或查機場接駁，可從這類比價站查詢。
錦糸町地區地圖 附近商店街、交通樞紐與餐廳分佈。

Day2｜2026/01/21（三） 鎌倉 → 橫濱 → 川崎

主題：歷史文化（鎌倉）× 港灣浪漫（橫濱）× 現代商業（川崎）

🗺️ 鎌倉地圖 🗺️ 橫濱地圖 🗺️ 川崎地圖
08:30
從錦糸町出發 → 前往鎌倉（交通建議）

推薦路線（方便、轉乘少）：

  • 錦糸町 → JR 線（總武快速 / Sobu Rapid）→ 東京（或品川）→ 轉乘JR橫須賀線/湘南新宿ライン→ 鎌倉：通常需在東京或品川轉乘橫須賀線。
  • 替代：錦糸町 → 地下鐵/都營轉乘至新宿/品川 → 乘橫須賀線直達鎌倉：視路線與時刻表選擇，轉乘次數相近。

車程總時間約 60–90 分鐘（視轉乘與列車類型）。

10:15–12:30
鎌倉 小町通（Komachi-dori）＋鶴岡八幡宮

小町通吃街、伴手禮與老店多，推薦試蒟蒻、和菓子與大佛燒。鶴岡八幡宮散步與參拜拍照（短程即可）。

建議出口/站口：鎌倉站 東口 → 步行至小町通入口。

12:40–14:00
移動 → 橫濱（午餐建議在橫濱）

從鎌倉搭 JR 或江之電前往橫濱（約 40–60 分鐘）。抵達後前往紅磚倉庫或中華街享用中式美食。

14:30–16:30
橫濱紅磚倉庫 × 元町商店街 × 中華街

紅磚倉庫逛文創小店、元町買特色小物、中華街提前排隊吃晚餐或小吃。

17:00–19:00
前往川崎 → LAZONA / KAWASUI / LA CITTADELLA

LAZONA 購物、KAWASUI（川崎水族館）適合晚間短遊，Le Cittadella（義大利風街區）可拍夜景、用餐。

20:30
返回錦糸町

晚間返回東京，約 21:30 抵達；視情況調整回程時間。

Day3｜2026/01/22（四） 銀座 → 東京車站一番街 → 表參道 / 原宿 → 澀谷

主題：銀座精品 × 藝術水族館 × 原宿潮流 × 澀谷夜景

🗺️ 銀座地圖 🗺️ 東京車站一番街 🗺️ 表參道/原宿 🗺️ 澀谷地圖
09:00
從錦糸町出發 → 抵達銀座

建議 09:00 後出門，避開 07:30–09:00 通勤高峰；銀座百貨多數 10:00 開門。

10:00–12:00
銀座：GINZA SIX / 銀座三越

逛重點百貨、旗艦店與伴手禮。有意參觀「銀座藝術水族館（Art Aquarium Museum）」建議預先購票。

12:30–13:30
移動 → 東京車站一番街（伴手禮 / 拉麵街）

可快速採買伴手禮或在拉麵街用餐。建議 45–60 分鐘。

14:30–16:30
移動 → 原宿 / 表參道（竹下通 / 表參道Hills）

原宿竹下通：潮流與年輕小店；表參道：精品與特色咖啡廳。

17:00–20:30
澀谷（Scramble Square / PARCO / 109 / Shibuya Sky）

若想看澀谷景觀建議前往 Shibuya Scramble Square / Shibuya Sky（視開放時間）。

Day5｜2026/01/24（六） 阿美橫丁（上野） → 晴空塔 → 錦糸町購物 → 前往成田

主題：庶民市場 × 晴空塔景觀 × 錦糸町購物收尾 → 機場出發

🗺️ 阿美橫丁 🗺️ 晴空塔 🗺️ 錦糸町
09:30–11:30
阿美橫丁（上野 Ameyoko）逛市集

市集販售藥妝、乾貨、小吃、伴手禮；適合早上出發避開擁擠。

12:30–14:30
移動 → 晴空塔（Skytree）午餐或觀景

晴空塔周邊商場有不少美食與伴手禮；上塔建議事先訂票並預留排隊時間。

15:00–17:30
錦糸町周邊購物（回飯店拿行李 / 收尾）

錦糸町有 PARCO、商店街與餐廳，可在此做最後採買或整理行李。

18:30–19:30
前往成田機場（TR875 回程 21:10 起飛）

建議至少在航班起飛前 2–2.5 小時抵達機場，計算交通與退稅時間，提早出發為佳。

交通建議 & 住宿說明
交通建議
  • 建議攜帶 Suica / PASMO：市內搭乘與小額消費皆方便。
  • 若行程多跨城（如鎌倉/橫濱/川崎），可評估購買一日券或使用 JR 東日本的短程票券。
  • 從錦糸町往成田建議預留 90–150 分鐘；視交通方式（Narita Express / 機場巴士 / 利木津）調整。
  • 前往鎌倉之建議：錦糸町 → JR（總武快速 / Sobu Rapid）→ 東京/品川 → 轉乘 JR 橫須賀線直達鎌倉（通常需在東京或品川轉乘一次）。
住宿（據點）

Tobu Hotel Levant Tokyo（錦糸町）交通方便、靠近地鐵與 JR，適合以東京市區為基地。若想早出發鎌倉，建議預留轉乘時間並避開早高峰。
